Detailed Description

template<typename T>
class CircularBuffer< T >

Circular buffer class to aid in computations for motion detection.

Template Parameters
TGeneric type so that the Circular Buffer can handle many different types of objects.

CircularBuffer contains the constructor, getter, setter, and cleanup functions for the CircularBuffer. The CircularBuffer is utilized within Camera's motion detection method to perform the frame comparison computations.
