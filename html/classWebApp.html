Detailed Description

WebApp is a self hosted web server that shows the status of the Smart Doorbell.

The Smart Doorbell takes in command line arguments to bind to an IP address. When running, use –docroot . –http-address 0.0.0.0 –http-port 8080 You can change the http address to your computer's IP address so that other devices on the network can access the webpage.
